# Project Name: Luxstore Mobile App

## Description

A cross-platform mobile application for Luxstore built with Expo. It delivers a native shopping experience on Android and iOS with features like push notifications, secure storage, and seamless navigation. The app integrates Stripe for mobile payments and uses TanStack Query for efficient data management.

## Tech Stack

- **Platform:** Expo / React Native
- **Language:** TypeScript
- **Navigation:** Expo Router
- **State/Data:** TanStack Query
- **Payments:** Stripe (React Native)
- **UI:** React Native Paper / Elements (implied or custom), Vector Icons
- **Features:** Expo Secure Store, Notifications, Haptics
